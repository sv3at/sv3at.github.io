# Simple Bible Reader

Offline Bible reader: JSON under `public/bible-data/`, no runtime Bible API.

![Parallel reading](docs/parallel-reading-screenshot.png)

## Quick start

```bash
npm install
npm run build:bible
npm run dev
```

Use the URL Vite prints. The app needs `http://` or `https://` (not `file://`).

```bash
npm test
```
